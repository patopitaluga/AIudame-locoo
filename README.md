# AIudame-locoo

Code helper for browser and terminal powered by open AI gpt 3.

------
## Setup
Create .env file:
```console
cp .env.example .env
```
Set your OPENAI_API_KEY variable, and PORT if required.

Install dependencies:
```console
npm install
```

------
## From terminal
```console
node aiudame your prompt
```

______

## As http server
```console
npm run start
```
